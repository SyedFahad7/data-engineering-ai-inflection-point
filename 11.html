Defined Consumers

AI Agents & Models Primary consumers requiring structured context, not just raw rows.

The Interface

Explicit Contracts Schemas are enforced. Breaking changes are treated like API failures.

Access Patterns Iceberg tables for bulk, high-speed APIs for inference.

Trust & SLAs

Reliability Guarantees 99.9% uptime for data freshness. Published incidents.

Semantic Meaning Metric definitions are versioned code, not wiki pages.

Ownership & Value

Domain Ownership Product teams own their data products, not a central data team.

Cost Awareness ROI is tracked per-product. "Is this data worth storing?"
